# FastMicCtrl - 快捷麦克风控制工具

FastMicCtrl 是一款专为 Windows 设计的轻量化麦克风控制程序。它允许用户通过全局快捷键瞬间开启或关闭系统中的所有麦克风，并提供直观的状态反馈。

## 🌟 主要功能

- **一键静音/取消静音**：通过全局快捷键控制全部录音设备。
- **极度轻量**：启动迅速，内存占用极低，完全绿色免安装。
- **系统托盘驻留**：程序启动后自动最小化到系统右下角托盘，不占用任务栏。
- **自定义快捷键**：用户可根据习惯自行录入组合键。
- **开机自启动**：支持用户选择是否随 Windows 启动。
- **多重状态反馈**：
  - **托盘图标**：绿色（开启）、红色（关闭）、黄色（状态不一致）、灰色（未检测到设备）。
  - **系统音效**：开启时播放清脆提示音（Asterisk），关闭时播放沉稳提示音（Hand）。
  - **悬浮提示**：鼠标悬停在托盘图标上可查看详细状态。

## 🚀 使用方法

1. **启动程序**：运行 `FastMicCtrl.exe`。
2. **默认快捷键**：`Ctrl + Alt + M`。
3. **切换状态**：按下快捷键或在托盘图标上点击右键选择“切换”。
4. **修改设置**：双击托盘图标，或右键选择“设置...”，在弹出窗口中录入新快捷键并保存。
5. **退出程序**：在托盘图标右键菜单选择“退出”。

## 🛠️ 开发者指南

### 项目结构
- `Program.cs`: 程序入口。
- `TrayApplicationContext.cs`: 核心逻辑，管理托盘图标、菜单及事件流。
- `MicrophoneController.cs`: 负责调用系统 API 控制录音设备。
- `HotkeyManager.cs`: 处理全局快捷键注册与监听。
- `SettingsForm.cs`: 设置界面的实现。
